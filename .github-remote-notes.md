# Bitbucket remote setup (OAuth token flow)

Use a token with **repository write + workspace/project admin** scope.

1. Set env:
   - `export BITBUCKET_TOKEN="..."`

2. Create repo via REST:
   - `curl -X POST -H "Authorization: Bearer $BITBUCKET_TOKEN" -H "Content-Type: application/json" https://api.bitbucket.org/2.0/repositories/agency-ops/openclaw-skills -d '{"scm":"git","is_private":true,"project":{"key":"OPS"}}'`

3. Push local `main`:
   - `git remote add origin git@bitbucket.org:agency-ops/openclaw-skills.git`
   - `git push -u origin main`

If repo already exists, skip step 2 and just run step 3.
